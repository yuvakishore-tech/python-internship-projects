Excel Report Generator

This project is a Python desktop application that generates an Excel report from a CSV file. It uses a graphical user interface built with Tkinter.

The application allows users to upload a CSV file, analyze numeric data, generate summary statistics, create pivot tables, and export a formatted Excel report automatically.

Technologies Used Python Pandas Matplotlib Openpyxl Tkinter

How to Run Install the required libraries. Run the main.py file. Upload a CSV file when prompted. Choose a location to save the Excel report.

Output The program generates an Excel file containing raw data, summary statistics, pivot tables, and an embedded chart.

This project is suitable for internship submissions and beginner-level Python learning.
